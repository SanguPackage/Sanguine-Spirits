Sanguine Spirits
================

_Where every sip runs deep._

Simple website with cocktail recipes. No ads, no fluff.

It's Sangu-Rita hour!

## Page Generation

```sh
ruby generate-cocktail-pages.rb

# Or register as git hook
cp pre-commit ./.git/hooks
```

## Images

Image prompts are generated by ChatGPT and the images by Midjourney.
They are 1024x1024 and get resized to 700x700 with `.\resize-cocktails.ps1`.
